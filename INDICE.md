## Cómo generar los Markdowns faltantes

```bash
# 1. Descargar los PDFs
python scripts/download_leyes.py --skip-existing

# 2. Convertir a Markdown
python scripts/batch_convert.py --skip-existing
```

```bash
python scripts/gen_indice.py
```
